# Run and deploy your StudentShifts app

This repo contains everything you need to run the demo locally.

## Run Locally

**Prerequisites:** Node.js 20+

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`

> Note: AI-powered helpers have been removed. Job descriptions and bios should now be written manually in the UI.
